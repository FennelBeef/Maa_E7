



# Maa_E7

<img src="D:../MaaDemo01\assets\伊蕾娜.jpg" alt="伊蕾娜" style="zoom:%;" border-radius:50px />

基于MaaFramework 的第七史诗小助手。图像技术 + 模拟控制，解放双手！  
由 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 强力驱动！

## 功能介绍

目前已有的功能：

- [x] 启动游戏

- [x] 每日一抽
- [x] 刷商店（誓约书签|神秘奖牌）
- [x] 每日商店领取免费体力，友情兑换体力，友情兑换旗帜
- [x] 自动竞技场
- [x] 获取每日宠物
- [x] 骑士团签到
- [x] 领取每日成就
- [x] 自动蹭远征

## 使用说明

下载地址：https://github.com/FennelBeef/Maa_E7/releases

### Windows

- 对于绝大部分用户，请下载 `Maa_E7-win-x86_64-v1.0.*.zip`
- 解压后运行 `MFAWPF.exe` 即可
